คำถามที่พบบ่อย

Tool ทุกตัวฟรีจริงไหม?
ใช่ครับ ฟรีทุก tool ไม่มีค่าใช้จ่ายซ่อนอยู่ ไม่ต้องสมัครสมาชิก ไม่ต้องให้อีเมล
ข้อมูลที่กรอกถูกเก็บไว้ไหม?
ไม่ครับ ข้อมูลทั้งหมด (เบอร์โทร วันเกิด ชื่อ) ประมวลผลในเบราว์เซอร์ของคุณเท่านั้น ไม่มีการส่งข้อมูลไปที่ server
ผลลัพธ์แม่นยำแค่ไหน?
ข้อมูลอ้างอิงจากตำราโหราศาสตร์ไทยและเลขศาสตร์ไทย นำเสนอเพื่อความบันเทิงและการศึกษา ไม่ใช่คำทำนายทางวิทยาศาสตร์
อยากให้เพิ่ม tool อะไรอีก?
ส่งความคิดเห็นมาได้เลยที่ hello@kambheerduang.com เรายินดีรับฟังและพิจารณาพัฒนาเพิ่มเติม
ข้อมูลในเว็บไม่ถูกต้อง จะแจ้งได้อย่างไร?
แจ้งมาที่ bug@kambheerduang.com พร้อมระบุ tool ที่ใช้ ข้อมูลที่กรอก และผลที่คาดว่าควรจะเป็น ทีมงานจะตรวจสอบและแก้ไขโดยเร็ว
